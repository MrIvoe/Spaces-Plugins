# Contributing to Spaces-Plugins

Thanks for contributing.

## Start Here

1. Read [Create A Plugin](docs/CREATE_A_PLUGIN.md).
2. Review [How It Works](docs/HOW_IT_WORKS.md).
3. Check [Plugins Offered](docs/PLUGINS.md) to avoid overlap.

## Pull Request Checklist

1. Validate plugin manifests.
2. Keep capability declarations accurate.
3. Test in Spaces with startup and settings persistence checks.
4. Update [Changelog](docs/CHANGELOG.md).

## Validation Command

```powershell
./scripts/validate-plugin-manifests.ps1
```

## Release Process

Follow [Release Guide](docs/RELEASE.md).
